## WordPress Optimization Tips

1. **Use a CDN** — Cloudflare (free) caches static content globally
2. **Install a caching plugin** — WP Rocket or W3 Total Cache
3. **Switch to PHP 8.x** — Significantly faster than PHP 7.x
4. **Use an object cache** — Redis or Memcached for database queries
5. **Image optimization** — Compress images before uploading (or use Smush)
