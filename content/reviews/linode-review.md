## Overview

Linode (acquired by Akamai in 2022) has long been a developer favorite. The documentation is still best-in-class, but the acquisition has brought some changes — some good, some not.

## Performance ($5/mo Plan)

- **Geekbench 6:** 1,089 (single-core)
- **Disk I/O:** 950 MB/s read, 510 MB/s write
- **Network:** 910 Mbps download, 840 Mbps upload
- **Uptime (30 days):** 99.96%

## Verdict

Still a solid choice for beginners who want great documentation. But the post-acquisition support dip means you might prefer DigitalOcean for similar pricing.
